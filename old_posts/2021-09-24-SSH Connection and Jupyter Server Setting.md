기존에 가지고 있던 LG gram 노트북을 개인용 서버로 만드는 작업을 기록한 것이다. 

# 목표
	MacBook Air M1에서 LG gram-2019을 원격으로 사용하기 

## SSH 설정

#### 서버로 작동할 윈도우에서 구성해야하는 작업
- OpenSSH Server 설치
- 22번 포트(SSH 디폴트) 방화벽 인바운드 규칙 설정 
- 공유기를 사용하는 경우, 공인 IP(공유기 IP)에서 사설 IP(공유기가 gram에 할당하는 IP)로 포트포워딩 

#### 클라이언트로 작동할 맥에서 구성해야하는 작업
- 딱히 없는 것처럼 보인다.

#### 클라이언트에서 서버로 접속하는 방법
- 터미널에서, '~ ssh [사용자이름]@[공인 IP]' 따옴표 안에 있는 문장을 입력한다. 여기서 사용자이름은 그램에서 사용하던 컴퓨터 이름이다. 이후 패스워드 입력이 요구될 때는 그램에서 사용하던 로그인 패스워드를 사용하면 된다. 

## Jupyter Server 셋업
이 부분은 SSH 설정과는 독립된 부분으로 볼 수 있다. 하지만, 서버에 접속한 뒤 Jupyter를 동작시켜야 할 상황이 있을 수 있다. 

#### 서버 jupyter notebook 세팅
- cmd창에 'jupyter notebook --generate-config' 입력하면, jupyter_notebook_config.py가 생성된다.
- Notepad와 같은 에디터를 이용하여 jupyter 동작을 설정한다. 포트, 보안, 작업 디렉토리, 외부 IP 진입 등 여러 옵션을 선택할 수 있다. 관련 설정은 다른 구글 서치를 통해 쉽게 확인 가능하다.

#### 방화벽 규칙 설정
- 위에서 설정된 포트를 마찬가지로 방화벽 인바운드 규칙에 넣어준다.
- 마찬가지로, 공유기를 사용하는 경우, 포트포워딩을 위와 같이 설정해준다. 

#### 내부/외부 IP에서 접속하는 방법
- 서버에서 Jupyter Lab이나 Notebook을 동작시킨 뒤, 'http://공인IP:포트번호'를 입력한다.
